# Ajedrez con Inteligencia Artifial

This project provides an API that implements the **alpha-beta pruning algorithm** for a chess game developed in Unity. It's important to note that the algorithm has been adapted to work for a four-player chess game, offering recommendations for the current player. These recommendations can be viewed below.
